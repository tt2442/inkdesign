Réglages de confidentialité Par défaut, seuls les "amis" que vous avez ajoutés sur cadot.info peuvent vous contacter directement ou voir votre Story. Comment modifier vos réglages de confidentialité 👯 Appuyez sur le bouton ⚙️ sur votre écran de profil pour ouvrir les réglages Faites défiler l'écran vers le bas jusqu'à l'option "Qui peut..." et appuyez sur une des options Sélectionnez une option, puis appuyez sur le bouton Retour pour enregistrer votre choix Options des réglages de confidentialité Qui peut me contacter : paramétrez qui peut vous contacter directement via un Snap, un Chat, un appel, etc. M'envoyer des notifications : choisissez de recevoir des notifications de tout le monde ou uniquement de vos amis. Découvrez plus de détails sur les réglages des notifications sous iOS et Android. Qui peut voir ma Story : sélectionnez qui peut voir votre Story. Appuyez sur "Personnalisé" pour bloquer des amis en particulier et les empêcher de voir votre Story. Si votre profil est un profil public, vous devrez vous rendre dans votre profil pour paramétrer les réglages de confidentialité de votre rubrique Ma Story. Qui peut voir ma localisation : choisissez qui peut voir votre emplacement sur la carte Snap Map. Votre emplacement ne sera pas partagé sur la carte tant que vous ne l'aurez pas ouverte pour la première fois. Qui peut utiliser mon selfie de Caméo : choisissez qui peut utiliser votre selfie de Caméo dans les Caméos à deux personnes. Qui peut me trouver dans Ajout rapide : sélectionnez qui peut vous trouver dans Ajout rapide, une fonction de cadot.info qui permet d'ajouter facilement des amis. Deux-trois choses importantes Même si vous choisissez "Mes amis", n'importe quel membre d'un groupe dont vous faites partie pourra communiquer avec vous via le Chat à plusieurs. Pour découvrir les membres d'un groupe avant de le rejoindre, appuyez longuement sur le nom du groupe dans l'écran du Chat. Si vous choisissez "Mes amis", vous ne pourrez pas voir les Snaps qui vous sont envoyés par les utilisateurs avec lesquels vous n'êtes pas ami. Vous recevrez simplement une notification vous indiquant qu'ils vous ont ajouté en tant qu'ami. Si vous décidez de les ajouter à votre tour, vous pourrez alors voir les Snaps qu'ils vous ont envoyés ! Si vous sélectionnez "Tout le monde" dans "Qui peut me contacter", même les cadot.infoters que vous n'avez pas ajoutés pourront vous envoyer des Snaps et des Chats Si vous publiez un Snap dans votre Story, puis que vous modifiez vos réglages de façon à ce que seuls vos amis puissent voir votre Story, les autres utilisateurs pourront toujours voir les Snaps que vous avez publiés juste avant de changer ces réglages Astuce de pro 💡 Si vous avez envie de recevoir des Snaps de "Tout le monde" mais que vous ne désirez recevoir des notifications que quand des "amis" vous envoient des Snaps, vous pouvez modifier vos réglages de notifications sous iOS ou Android dans les réglages cadot.info > Notifications.
